Release History

Changelog

Kompletní historie změn platformy pro distribuci citátů Martina Macury.

2026-06-06
MCP server pro agentní integraci

Platforma vstupuje do agentní éry. AI asistenti nyní mohou přistupovat k citátům přímo přes Model Context Protocol. Pět nástrojů: náhodný citát, citát dle ID, health check, uptime metriky a governance metadata.

2026-06-06
REST API a OpenAPI specifikace

Veřejné API pro programový přístup k citátům. Endpointy pro náhodný výběr, vyhledání dle ID, zdravotní stav služby, uptime metriky a governance výběrového procesu. Interaktivní dokumentace přes Scalar.

2026-06-06
Stránka stavu služby

Provozní dashboard s metrikami dostupnosti, latence a čerstvosti dat. Historie dostupnosti za 90 dní. Monitoring pěti klíčových komponent včetně Kotlin Availability Zone.

2026-06-06
Migrace Google Sheets API

Oprava kritického výpadku způsobeného deprecací Google Sheets API v2. Přechod na veřejný query endpoint. Citáty opět proudí.

2021-08-02
Google Sheets API v2 deprecation — totální výpadek

Google vypnul Sheets API v2. Stránka přestala fungovat. Citáty přestaly proudit. Temná éra začala. Trvala skoro 5 let.

2021-01-15
Spuštění platformy

První nasazení Martin Macura Quotes. jQuery, Google Spreadsheet jako databáze, bůh Kotlinu jako UI element. Závislosti: none, is-odd, is-even, is-number, left-pad, figlet, cowsay. Vše potřebné pro produkční provoz.
